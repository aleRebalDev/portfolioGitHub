Discovering Javascript

I studied a professional cycle of Telecommunications and IT Systems in Spain. It was from 2010 to 2013. In 2018 I started studying again, online on the Open Webinars platform, I was focused on networks, and that is when I discovered programming.

After studying basic programming and looking for a language I like, I took a Javascript course in 2019 on the Udemy platform at the beginning and I have been studying a lot since then in different places.

What I like about Javascript

I enjoy solving challenges, I like to find a way to obtain a satisfactory result. Javascript is very versatile, I can use it both in backend and frontend, O.O.P and functional programming (learning FP is one of my goals in the future).

I can not really compare it to other languages ​​because I have not used another one (I started learning Python, but changed it to Javascript). What I like are its methods for manipulating data, objects and literals. I am learning a lot from this and I enjoy it very much.

I use it to develop my backend in Node, and on the frontend I am using TypeScript because of Angular, but I like to use pure Javascript too, be it on the frontend, solving algorithms or wherever I need it.
